# 🎓 クラス配属システム

必修科目のクラス配属を自動化するローカル動作のWebアプリです。  
学生の希望に基づいてクラスを割り当て、定員超過時は抽選を行います。

## 特徴

- **完全ローカル動作** — 入力データは外部に送信されません
- **xlsx 入出力** — 既存の名簿データをそのまま使用可能
- **再現性のある抽選** — シード値を指定することで同じ結果を再現可能
- **インストール不要** — ブラウザで HTML ファイルを開くだけで動作

## 対象規模

- 配属対象者：最大数千名（デフォルト設定：290名）
- クラス数：任意（デフォルト：8クラス）
- 定員：クラスごとに個別設定可能

## 使い方

### 1. テンプレートの取得

アプリを開き「📄 テンプレートをDL」ボタンを押すと、入力用の xlsx テンプレートをダウンロードできます。

### 2. データの準備

テンプレートの `学生データ` シートに以下の列を埋めてください。

| 列名 | 内容 | 例 |
|------|------|----|
| `学籍番号` | 学生の ID | `S2024001` |
| `氏名` | 学生の氏名 | `山田 太郎` |
| `第1希望` | 第1希望のクラス名 | `クラス1` |
| `第2希望` | 第2希望のクラス名 | `クラス3` |
| `第3希望` | 第3希望のクラス名 | `クラス5` |

> **注意：** クラス名はアプリ内の設定と完全に一致させてください（例：`クラス1`）。

### 3. 設定の確認

- **クラス数** と **各クラスの定員** をサイドバーで調整
- **シード値** を固定すると抽選結果を後から再現できます（空欄の場合はランダム）

### 4. 配属の実行

xlsx ファイルをアップロードし「▶ 配属を実行」を押すと即座に結果が表示されます。

### 5. 結果のダウンロード

「⬇ 結果をxlsxでDL」で以下のシートを含む xlsx ファイルを取得できます。

| シート名 | 内容 |
|----------|------|
| 配属結果一覧 | 全学生の配属クラスと配属方法 |
| クラス別集計 | クラスごとの人数・希望別内訳 |
| クラス1〜N | クラスごとの名簿 |

## 配属アルゴリズム

```
1. 全学生を第1希望のクラスに仮配属する

2. 定員を超過したクラスでは抽選を実施
   → 当選した学生：第1希望に確定
   → 落選した学生：次のステップへ

3. 落選した学生を第2希望 → 第3希望の順に配属
   → 希望先に空きがあれば確定
   → 空きがなければ次のステップへ

4. 第3希望まで決まらなかった学生を、
   空きのあるクラスにランダムで配属
```

## 動作環境

モダンブラウザ（Chrome / Firefox / Edge / Safari）で動作します。  
インターネット接続は Google Fonts の読み込みにのみ使用されます（オフラインでも機能しますが、フォントが変わります）。
